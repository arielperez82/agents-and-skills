# Transformation Checklist

Use this checklist for every script-to-article transformation. Each step must pass before delivering the output.

## Pre-Transformation

- [ ] Source script read in full — no skimming
- [ ] Source word count recorded
- [ ] Target compression ratio selected (3x, 4x, or 5x based on source density)
- [ ] Target output word count calculated (source / compression ratio)
- [ ] Story boundaries identified (single story or multi-story segmentation needed?)

## Factual Extraction

- [ ] Every factual claim listed (numbers, dates, names, events, outcomes)
- [ ] Every direct quote captured with attribution
- [ ] Every source/reference mentioned in the script noted
- [ ] Every organization and person mentioned logged
- [ ] Causal claims identified ("X caused Y," "X led to Y") — these are high-risk for distortion

## Verbal Artifact Removal

- [ ] Filler words removed (you know, like, I mean, basically, actually, right?)
- [ ] Delivery cues removed (now let's talk about, here's the thing, as I mentioned)
- [ ] Emphasis repetition condensed to single statement
- [ ] False starts cleaned up
- [ ] Audience address phrases removed (for those of you, if you've been following)
- [ ] Conversational transitions replaced with written transitions

## Restructuring

- [ ] Lead rewritten for reading (news-first, not build-up)
- [ ] Related facts grouped (not scattered as in spoken delivery)
- [ ] Paragraph breaks at topic shifts (not breathing pauses)
- [ ] Logical flow verified (each paragraph follows naturally from previous)
- [ ] Closing provides context or implications (not "thanks for watching")

## Compression Verification

- [ ] Output word count within target range
- [ ] Compression ratio calculated (source / output)
- [ ] If < 3x: identify what can still be cut (commentary before facts)
- [ ] If > 5x: verify no material facts were lost

## Fidelity Check (Critical)

- [ ] Every factual claim in output maps to a specific passage in source
- [ ] No facts added that don't appear in source
- [ ] No numbers changed or rounded without noting
- [ ] All attributions preserved ("according to X" not dropped)
- [ ] All direct quotes are verbatim from source
- [ ] Causal claims not strengthened (correlation → causation drift)
- [ ] No editorializing added (opinions not in the source)

## Multi-Story Segmentation (when applicable)

- [ ] Each story is self-contained (readable without other stories)
- [ ] Context that was implicit in show flow made explicit in each story
- [ ] No facts from Story A leaked into Story B
- [ ] Each story has: headline, one-line summary, body
- [ ] Story count matches topic count from source

## Final Quality

- [ ] Grammar and spelling checked
- [ ] No orphaned references ("as mentioned above" with no above)
- [ ] Headline is accurate and not clickbait
- [ ] Tone is neutral reporting (not persuasive, not editorial)
