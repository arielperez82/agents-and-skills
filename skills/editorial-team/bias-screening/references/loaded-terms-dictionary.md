# Loaded Terms Dictionary

Common loaded terms organized by domain. For each loaded term, use the neutral alternative in editorial reporting.

**Usage:** Scan article text against these terms. If a match is found, evaluate context — some terms are neutral in certain contexts (e.g., "crisis" is loaded for a policy disagreement but neutral for an actual humanitarian crisis). Flag only when usage is loaded.

---

## Politics & Government

| Loaded Term | Neutral Alternative | Notes |
|-------------|-------------------|-------|
| regime | government, administration | "Regime" implies illegitimacy |
| scheme | plan, program, proposal | "Scheme" implies deception (in US English) |
| radical | [describe the actual position] | Subjective; one person's radical is another's mainstream |
| moderate | [describe the actual position] | Implies a value judgment about what is reasonable |
| extremist | [describe the actual position] | Label that substitutes for description |
| crackdown | enforcement action, restrictions | "Crackdown" implies both force and approval |
| slam / blast (criticism) | criticized, objected to, said | Exaggerates the intensity of disagreement |
| admit | said, stated, acknowledged | "Admit" implies wrongdoing |
| claim (for one side only) | said, stated | "Claim" implies doubt; use consistently or not at all |
| reckless | [omit or describe specific risk] | Value judgment |
| bold | [omit or describe specific scope] | Value judgment |
| common-sense (policy) | [describe the policy] | Implies disagreement is nonsensical |

## Economics & Business

| Loaded Term | Neutral Alternative | Notes |
|-------------|-------------------|-------|
| slashed (budgets, prices) | reduced, cut | "Slashed" implies violence/destruction |
| soared / skyrocketed | increased, rose [by X%] | Exaggeration; use the actual number |
| plummeted / collapsed | decreased, fell [by X%] | Exaggeration; use the actual number |
| crisis (policy disagreement) | dispute, disagreement | Reserve "crisis" for genuine emergencies |
| bailout | financial assistance, rescue package | "Bailout" implies irresponsibility |
| handout | benefit, assistance, program | "Handout" implies undeserved |
| job-killing | [describe the actual impact] | Causal claim that may not be supported |
| trickle-down | supply-side economics | "Trickle-down" is a political label, not an economic term |

## Social Issues

| Loaded Term | Neutral Alternative | Notes |
|-------------|-------------------|-------|
| illegal alien | undocumented immigrant, unauthorized immigrant | Legal status ≠ personhood label |
| pro-life / pro-choice | abortion opponent / abortion-rights supporter | Both "pro-" terms are advocacy labels |
| gun control / gun rights | firearm regulation / firearm policy | Both terms carry ideological weight |
| woke | [describe the actual position or policy] | Undefined, used as both compliment and insult |

## Context Rules

1. **Domain matters:** "Crisis" is loaded for a budget disagreement but neutral for a natural disaster. Evaluate in context.
2. **Direct quotes are exempt:** If a source uses loaded language in a direct quote, preserve it — but don't adopt it in your own text.
3. **Consistency test:** If you use "claim" for one side, use it for both sides — or use "said" for both.
4. **Specificity over labels:** When in doubt, describe the specific thing rather than using a label. "The bill would increase the top marginal tax rate from 37% to 39.6%" is always better than "the tax hike" or "the tax reform."
