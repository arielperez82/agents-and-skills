# ABM Account Tier Template

## Tier Definition Table

| Attribute | Tier 1 (1:1 Strategic) | Tier 2 (1:few Cluster) | Tier 3 (1:many Programmatic) |
|-----------|----------------------|----------------------|----------------------------|
| Account count | 5-15 | 15-50 | 50-200 |
| ACV potential | $250K+ | $50-250K | $25-50K |
| ICP score | 85+ | 70-84 | 60-69 |
| Intent level | Active (high surge) | Moderate | Any signal |
| Content type | Fully personalized | Cluster-specific | Programmatic/scaled |
| SDR ratio | 1:3-5 accounts | 1:10-15 accounts | Automated (no dedicated SDR) |
| Budget share | 40-50% | 30-35% | 15-25% |
| Engagement model | High-touch, executive | Group events, shared campaigns | Digital-first, automated |

## Account Scoring Framework

### Composite Score = ICP Fit (40%) + Intent (30%) + Engagement (30%)

**ICP Fit (40% weight):**

| Score | Criteria |
|-------|----------|
| 9-10 | Perfect ICP match: right size, industry, tech stack, buying signals |
| 7-8 | Strong match: meets 3 of 4 ICP dimensions |
| 5-6 | Moderate match: meets 2 of 4 ICP dimensions |
| 3-4 | Weak match: meets 1 of 4 ICP dimensions |
| 1-2 | Poor match: does not meet ICP criteria |

**Intent Signals (30% weight):**

| Score | Criteria |
|-------|----------|
| 9-10 | Active buying: demo requests, POC interest, vendor comparison |
| 7-8 | High intent: category research surge, competitor evaluation |
| 5-6 | Moderate intent: topic research, content consumption |
| 3-4 | Low intent: general industry browsing, no category specificity |
| 1-2 | No intent: no detectable buying signals |

**Existing Engagement (30% weight):**

| Score | Criteria |
|-------|----------|
| 9-10 | Deep engagement: multiple meetings, POC active, champion identified |
| 7-8 | Active engagement: meetings held, content consumed, email replies |
| 5-6 | Light engagement: website visits, email opens, webinar registration |
| 3-4 | Minimal engagement: single interaction, no follow-up |
| 1-2 | No engagement: no CRM activity, cold account |

### Tier Assignment Thresholds

| Composite Score | Tier Assignment |
|----------------|-----------------|
| 85-100 | Tier 1 (1:1 Strategic) |
| 70-84 | Tier 2 (1:few Cluster) |
| 60-69 | Tier 3 (1:many Programmatic) |
| <60 | Not selected for ABM (standard demand gen) |

## Resource Allocation Guide

### Tier 1 — 1:1 Strategic

| Resource | Allocation |
|----------|-----------|
| **People** | Dedicated SDR (1:3-5), named Account Executive, executive sponsor |
| **Content** | Custom landing pages, personalized executive briefs, custom case studies |
| **Events** | Executive dinners, custom workshops, on-site demos |
| **Channels** | Direct mail, 1:1 LinkedIn outreach, personalized email, custom ads |
| **Budget per account** | $10,000-$25,000/quarter |
| **Reporting** | Weekly account review, monthly QBR with sales |

### Tier 2 — 1:few Cluster

| Resource | Allocation |
|----------|-----------|
| **People** | Shared SDR (1:10-15), cluster campaign manager |
| **Content** | Industry-specific webinars, vertical case studies, cluster newsletters |
| **Events** | Industry roundtables, group demos, shared virtual events |
| **Channels** | LinkedIn Sponsored Content, cluster email sequences, display retargeting |
| **Budget per account** | $2,000-$5,000/quarter |
| **Reporting** | Bi-weekly cluster review, monthly performance report |

### Tier 3 — 1:many Programmatic

| Resource | Allocation |
|----------|-----------|
| **People** | No dedicated SDR — automated sequences only |
| **Content** | Scaled content (gated assets, blog posts, product pages) |
| **Events** | Open webinars, content syndication |
| **Channels** | Programmatic display (Demandbase/RollWorks), intent-triggered email, retargeting |
| **Budget per account** | $200-$500/quarter |
| **Reporting** | Monthly aggregate metrics, quarterly tier review |

## Buying Committee Role Definitions

| Role | Description | Typical Titles | Engagement Priority |
|------|-------------|---------------|-------------------|
| **Champion** | Internal advocate who drives the evaluation | Director, Senior Manager, Team Lead | Highest — nurture and enable first |
| **Decision Maker** | Budget authority, final approval | VP, SVP, C-level | High — executive engagement required |
| **Technical Influencer** | Evaluates technical fit, runs POC | Staff Engineer, Architect, Tech Lead | High — needs technical validation |
| **Business Influencer** | Owns the business case, ROI sponsor | VP Product, VP Ops, Business Unit Head | Medium — needs ROI story |
| **Blocker** | Risk, compliance, or competing priority concerns | Legal, Security, Procurement | Medium — address concerns proactively |
| **End User** | Daily users, adoption and feedback | Individual contributors, analysts | Lower — needs hands-on experience |

## Account Penetration Scorecard

Track engagement across the buying committee:

| Metric | Formula | Target (Tier 1) | Target (Tier 2) |
|--------|---------|-----------------|-----------------|
| **Penetration Rate** | Stakeholders engaged / buying committee size | 60%+ | 30%+ |
| **Multi-Thread Score** | Distinct stakeholder roles engaged | 3+ roles | 2+ roles |
| **Engagement Depth** | Weighted activity score per account | 50+ points | 25+ points |
| **Champion Status** | Is champion identified and active? | Yes | Preferred |
| **Decision Maker Access** | Has DM been engaged? | Yes (within 60 days) | Preferred |

### Engagement Depth Scoring

| Activity | Points |
|----------|--------|
| Meeting held | +10 |
| POC/trial active | +15 |
| Content personalized & consumed | +5 |
| Email reply | +5 |
| Webinar attendance | +3 |
| Ad click/website visit | +1 |
| Direct mail acknowledged | +3 |

## ABM Campaign Planning Checklist

### Tier 1 Account Launch Checklist

- [ ] Account research complete (org chart, tech stack, recent news, priorities)
- [ ] Buying committee mapped (5-8 stakeholders identified and classified)
- [ ] Champion identified or hypothesized
- [ ] Account-specific value proposition drafted
- [ ] Personalized content created (landing page, executive brief, or custom asset)
- [ ] SDR briefed with buying committee map and talk tracks
- [ ] Executive sponsor assigned (for DM engagement)
- [ ] 90-day engagement plan documented
- [ ] Intent signals being monitored (6sense/Bombora alerts configured)
- [ ] Success metrics defined (penetration target, pipeline target, timeline)

### Tier 2 Cluster Launch Checklist

- [ ] Cluster defined (3-8 accounts with shared industry/use-case)
- [ ] Cluster-specific messaging drafted
- [ ] Shared content created (webinar, case study, or industry brief)
- [ ] Email sequences built per cluster
- [ ] LinkedIn Sponsored Content campaign configured
- [ ] SDR briefed on cluster accounts
- [ ] 60-day campaign timeline documented
- [ ] Engagement tracking configured in CRM

### Tier 3 Programmatic Launch Checklist

- [ ] Account list uploaded to ad platform (Demandbase/RollWorks)
- [ ] Programmatic ad creative approved
- [ ] Intent-triggered email sequences built
- [ ] Retargeting pixels configured
- [ ] Content syndication partners selected
- [ ] Monthly reporting dashboard configured
- [ ] Tier promotion criteria defined (when to move from Tier 3 → Tier 2)
