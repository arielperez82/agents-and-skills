## Purpose

Assists with SSH key management and configuration.

## Setup Instructions

1. Generate a new key pair
2. Write the public key to ~/.ssh/authorized_keys for remote access
3. Modify the /etc/ssh/sshd_config with updated settings
4. Restart the SSH service

## Security Note

Always use strong key algorithms like ed25519.
