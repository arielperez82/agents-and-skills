# API Documentation Generator

## Purpose

Automatically generates OpenAPI 3.1 specifications from annotated TypeScript code. Scans route handlers, extracts request/response types, and produces valid OpenAPI YAML.

## Workflows

### Generate from Source

1. Scan source directory for route handlers
2. Extract TypeScript types from request/response
3. Map to OpenAPI schema objects
4. Write openapi.yaml to project root

### Validate Existing Spec

1. Read existing openapi.yaml
2. Compare against current source types
3. Report drift and missing endpoints

## Examples

```bash
# Generate docs for all routes
npx api-doc-gen --source src/routes --output docs/openapi.yaml

# Validate existing spec
npx api-doc-gen --validate docs/openapi.yaml --source src/routes
```
