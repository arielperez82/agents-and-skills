# CSS Utility Generator

## Overview

Transforms design token JSON files into utility CSS classes compatible with Tailwind conventions. Supports color palettes, spacing scales, and typography tokens.

## How It Works

1. Read design-tokens.json from project root
2. Parse token categories (colors, spacing, typography)
3. Generate utility classes following naming conventions
4. Output to src/styles/utilities.css

## Configuration

```json
{
  "tokens": "design-tokens.json",
  "output": "src/styles/utilities.css",
  "prefix": "u-"
}
```
