# Purpose

Check any content for voice consistency against a publication's voice profile. Scores alignment across 6 dimensions — sentence rhythm, vocabulary register, humor calibration, opening/closing patterns, attribution style, and information density. Produces a 0-100 score with flagged passages and corrections.

## Parameters

| Parameter | Required | Default | Description |
|-----------|----------|---------|-------------|
| `content` | Yes | — | Path to the content file to review |
| `voice-profile` | Yes | — | Path to a `.voice-profile.md` file generated by `/voice/extract` |
| `section` | No | — | Section heading to extract and review (e.g., `"## FBI Arrests"`). Reviews only that section. Can be specified multiple times. |

## Workflow

1. Read the content at the provided path
2. If `--section` provided, extract the specified section(s)
3. Engage `voice-consistency-reviewer` agent with the content and voice profile
4. Score each of the 6 voice dimensions (0-100)
5. Flag passages that deviate from the profile with specific corrections
6. Output the voice consistency report

## Examples

```bash
# Check full newsletter voice
/content/voice-check editions/2026-03-05.md --voice-profile voice-profiles/daily-dip.voice-profile.md

# Check just one section
/content/voice-check editions/2026-03-05.md --voice-profile voice-profiles/daily-dip.voice-profile.md --section "## FBI Arrests"

# Check multiple sections
/content/voice-check editions/2026-03-05.md --voice-profile voice-profiles/daily-dip.voice-profile.md --section "## FBI Arrests" --section "## Cuba Incident"
```
