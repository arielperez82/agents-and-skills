# Waste Snake

The waste snake is a living record of waste observations. Anyone (agent or human) can append observations here. The `agile-coach` periodically reviews and aggregates them into ledger entries.

**How to add:** Run `/waste/add "description"` or append manually using the format below.

**Format:**

```markdown
### YYYY-MM-DD HH:MM

[Free-text description of what frustrated, bored, or delayed you]
```

**Reference:** See `skills/delivery-team/waste-identification/SKILL.md` for the full practice description.
